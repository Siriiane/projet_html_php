Portée des variables (=scope)

  • une variable déclarée dans une fonction n'est accessible que dans cette fonction. On dit que c'est une variable locale à la fonction.
  • Une variable déclarée avec le mot clé var est ubne variable globale. Elle est accessible partout
  • Une variable déclarée avec le mot-clé let n'est accessible que dans le bloc ou elle à été déclarée. Un bloc est déliùmité par des (accolades, curly bracket en anglais). On ne peux pas redéclatre une variable déclarer avec let dans le même bloc .
  • le mot clé const permet de déclarer des constantes. une constantes est une variable dont la valeur ne peut pas être modifiée. Comme pour let, const a une portée de bloc.
