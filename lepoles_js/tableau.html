TABLEAU : Array

un tableau est une variable ui va contenir plusieurs valeurs. Pour déclarer un tableau, on utilise la synthaxe suivante

let nomDeLAVAriable = [ valeur1, valeur2, ...]

Pour accéder à une valeur de ce tableau, on va utiliser son indice, c'est a dire sa position dans le tableau.

nomDeLaVariable[indice]

⚠ la numérotation des indices commences à 0

Ajouter une valeur dans un tableau

On va utiliser la fonction push de l'objet tableau.

Syntaxe : tableau.push(valeurAajouter);

Supprimer la dernière valeur d'un tableau

La fonction pop de l'objet tableau supprime la dernière valeur du tableau.

Syntaxe : tableau.pop();
